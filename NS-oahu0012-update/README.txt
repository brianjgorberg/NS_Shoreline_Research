# Title: Satellite-Derived Shorelines for O‘ahu,
Addition: Planetscope data only 2017-2025

Version: v1.0

**Creators:**
Anna Mikkelsen, Joel Nicolow, Richelle Moskvichev
Coastal Research Collaborative, University of Hawai‘i at Mānoa

Date of Creation: Nov 2025
Date Range of Observations: 2017–2025

**Geographic Coverage:**

- Region: North Shore of Oahu (litoral cell 0012) based on transects generated for CoSMoS-COAST (version 2023-11-27)

**Dataset Description:** 

This dataset contains satellite-derived shoreline (SDS) transect intersections for littoral cell 0012 - updated to include 2023-2025 data using only planetscope and coastvision. The data has been tidally corrected and have major outliers removed.
